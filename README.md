# Pokemon Game

1. Execute `yarn` to install all dependencies.
```
$ yarn install
```
2. Compiles and hot-reloads for development.
````
$ yarn serve
````
3. Run unit testing
```
$ yan test:unit
```
4. To compile and minifiles for production build.
````
$ yarn build
````
